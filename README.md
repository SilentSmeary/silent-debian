# silent-debian
## One Line Command
```
curl -fsSL https://raw.githubusercontent.com/SilentSmeary/silent-debian/main/setup.sh | sh
```
## Usage
```
git clone https://github.com/SilentSmeary/silent-debian
cd ./silent-debian
bash setup.sh
```
